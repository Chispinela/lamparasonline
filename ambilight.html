¿Que es el ambilight?

El ambilight es un tipo de iluminación que se basa en ambientarse con un aparato tecnológico. Esto significa que gracias a esta nueva tecnologia puedes poner leds ambientados con el que esta ocuriendo en tu pantalla o monitor.

¿Como se instala el ambilight?

Para instalar el ambilight lo que normalmente se suele utiliza es una tira led ya que tiene una muy buena compatibilidad con el dispositivo que utilizaremos. A continuación vamos a selecionar los materiales necesarios y recomendados después vamos a redactar los pasos a seguir. Recuerda leer los pasos antes de comprar el kit o la tira led.

Materiales necesarios:

Tira led: Necesitaremos una tira led para poner las luces detrás del monitor o pantalla.

Controlador led: Para poder controlar la tira led desde el ordenador hay que usar un aparato para enviar la información y cambiar el color.

Programa: Hay muchos programas de ordenador para poder controlar la tira led pero el que recomendamos es ambibox ya que es el mas conocido y ademas gratis.

Materiales recomendados

Kit de Aliexpress: Para usar ambilight recomendamos un kit que se encuentra en Aliexpress si quieres ver el modelo pulsa aquí.

Pasos a seguir:

A continuación vamos a redactar cuales son los pasos para poder montar nuestro kit de ambilight comprado en Aliexpress.

1= Calculamos las pulgadas de nuestra pantalla o monitor para poder elegir la longitud de nuestra tira led. Para poder calcular la longitud de tira led que necesitamos nos guiaremos con la imagen inferior.

2= Colocamos la tira led en la parte trasera de nuestro monitor como podemos ver en la imagen inferior al texto. Si sobra cinta de la tira led la podemos cortar con unas tigeras por la parte que indica el producto.

3= Conectamos el dispositivo ambilight con el cable de alimentación de la tira led. A continuación volvemos a conectar el dispositivo ambilight esta vez con el ordenador por puerto usb.

4= Instalamos el programa de Ambibox con el cúal podremos controlar la tira led con el pc.

5= En el programa selecionamos el puerto en el cual hemos conectado nuestro dispositivo ambilight y ya habremos conectado nuestra tira led con el pc. Si quieres tener mas información sobre el ambilight puedes ver este video.
